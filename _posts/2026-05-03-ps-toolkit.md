# 🛠️ PS.Toolkit - PeekSecurity

## 🔐 Um toolkit educacional de segurança e automação no Termux

O **PS.Toolkit** é uma ferramenta desenvolvida pela equipe **PeekSecurity** com foco em educação em segurança cibernética, automação de tarefas e utilitários para Termux.

📦 Repositório oficial:  
https://github.com/PSecurity/pstoolkit

Este projeto foi desenvolvido exclusivamente para fins educacionais.

- Use apenas em ambientes controlados

<div class="divider"></div>

## 🚀 O que é o PS.Toolkit?

O PS.Toolkit funciona como um conjunto de utilitários modulares para Termux, permitindo:

- Automação de comandos
- Ferramentas de reconhecimento
- Auxílio em análise de rede
- Base extensível para novos módulos
