## 🚀 O que é o PS.Nmap?

**PS.Nmap** é uma ferramenta de reconhecimento de rede, construída para facilitar o uso do Nmap em dispositivos móveis e ambientes de baixo recursos, como o **Termux**. Desenvolvida para entusiastas da segurança, pentesters e hackers, a ferramenta automatiza o processo de escaneamento de redes, permitindo realizar testes rápidos e eficientes.

Este script simplifica o uso do Nmap, torna a execução de scans mais rápida e oferece uma interface fácil para configurar e executar comandos de reconhecimento.

## ⚡ Funcionalidades

O PS.Nmap oferece diversas funcionalidades para tornar o processo de recon mais eficiente, incluindo:

- **Automatização de escaneamentos Nmap**
- **Facilidade de configuração no Termux**
- **Execução simples e rápida de escaneamentos**
- **Compatibilidade com Termux e Kali Linux**
- **Relatórios de resultados claros**

## ⚙️ Instalação

Para instalar o **PS.Nmap** no seu dispositivo Android (via Termux) ou em Kali Linux, siga as instruções abaixo.

### 📱 Instalando no Termux

```bash
pkg update -y && pkg upgrade -y
pkg install git nmap -y
git clone https://github.com/PSecurity/ps.nmap.git
cd ps.nmap
chmod +x ps.nmap

### 💻 Instalando no Kali Linux

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install git nmap -y
git clone https://github.com/PSecurity/ps.nmap.git
cd ps.nmap
chmod +x ps.nmap

### 🧠 Como Usar

O PS.Nmap foi desenvolvido para facilitar o uso do Nmap no Termux/Kali. Após a instalação, basta rodar o script com os parâmetros desejados.

Exemplo básico de execução:

./ps.nmap -T4 -A -v <target_ip>

Onde:

-T4: define o nível de intensidade do scan.

-A: habilita a detecção de versão, scripts, etc.

-v: mostra a saída detalhada.


### 🔥 Recursos e Testes

Aqui estão alguns dos principais comandos e usos para o PS.Nmap:

### 🌐 Escanear uma Rede

./ps.nmap -sn 192.168.1.0/24

### 🚨 Testar uma Porta Específica

./ps.nmap -p 80,443 <target_ip>

### 🛡️ Detectar Vulnerabilidades

./ps.nmap -sV --script vuln <target_ip>

### ⚠️ Dicas e Melhorias

Sempre atualize o Nmap: Certifique-se de ter a versão mais recente do Nmap instalada.

Evite scans em redes não autorizadas: Lembre-se de sempre obter permissão para realizar testes de penetração.

Configurações avançadas: Utilize as opções avançadas do Nmap para personalizar os seus scans conforme necessário.


### 🛡️ Dica de Segurança

Ao utilizar PS.Nmap, esteja ciente dos riscos de escaneamento em redes sem permissão. Certifique-se de que está operando dentro da legalidade e conforme as regras de uso da rede.
