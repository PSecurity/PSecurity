## 🚀 O que é

O **PS.Ngrok** é um script automatizado que instala e configura o Ngrok ou Cloudflare Tunnel de forma simples no Termux e Kali Linux.

<hr class="divider">

## ⚡ Instalação

<div class="code-box">
<button onclick="copyCode(this)">Copiar</button>
<pre><code>git clone http://github.com/PSecurity/ps.ngrok"</code></pre>
</div>

<hr class="divider">

## 🧠 O que esse script faz

* Detecta automaticamente o ambiente (Termux ou Kali)
* Corrige erros de arquitetura do Ngrok
* Instala versão compatível automaticamente
* Oferece opção de usar Cloudflare Tunnel
* Configura o authtoken do Ngrok

<hr class="divider">

## ⚙️ Como usar

### 🔓 Ngrok

<div class="code-box">
<button onclick="copyCode(this)">Copiar</button>
<pre><code>ngrok http 8080</code></pre>
</div>

### ☁️ Cloudflare

<div class="code-box">
<button onclick="copyCode(this)">Copiar</button>
<pre><code>cloudflared tunnel --url http://localhost:8080</code></pre>
</div>
